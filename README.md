## ✨ Overview

MockMind is a full-stack AI interview preparation platform designed to help students, job seekers, and professionals prepare for real-world interviews with confidence.

Upload your resume, generate AI-powered interview questions tailored to your profile, practice technical and HR rounds, and receive intelligent feedback that helps you improve after every session.

## 🎯 Key Features

- 📄 Resume-Based Interview Question Generation
- 🤖 AI-Powered Technical & HR Interview Simulation
- 📊 Intelligent Feedback & Performance Evaluation
- 💳 Credit-Based Usage System
- 💰 Razorpay Payment Integration

## 🚀 How It Works

1. Login using Google Authentication
2. Upload your Resume (PDF)
3. AI analyzes your profile
4. Generate personalized interview questions
5. Practice Technical & HR interviews
6. Receive AI-generated feedback
7. Purchase credits when needed

## ⚙️ Environment Variables

### Client (.env)

```env
VITE_FIREBASE_APIKEY=your_firebase_api_key
VITE_RAZORPAY_KEY_ID=your_razorpay_key_id
```

### Server (.env)

```env
PORT=8000
MONGODB_URL=your_mongodb_url
JWT_SECRET=your_jwt_secret
OPENROUTER_API_KEY=your_openrouter_api_key
RAZORPAY_KEY_ID=your_razorpay_key_id
RAZORPAY_KEY_SECRET=your_razorpay_key_secret
```
